## Importance of js

- a language used to build interactivity to webpages.
- JavaScript is one of the most widely available and frequently used methods for interacting with the browser on the client side, the language became immensely popular.
- only computer language that allows us to directly interact with our web pages dynamically on the client.
-  So removing JavaScript from the browser would effectively render millions of websites completely useless.
